# Support

Sightline is an early prototype and does not provide public support guarantees.

## Setup Help

Use GitHub issues for setup questions, local demo problems, RTSP issues, missing model errors, or Docker problems.

Before opening an issue, check:

- [README.md](README.md)
- [docs/LOCAL_SETUP.md](docs/LOCAL_SETUP.md)
- [docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md)

## Security Issues

Do not report security vulnerabilities in public issues.

Follow [SECURITY.md](SECURITY.md).
