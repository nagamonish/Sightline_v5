# Changelog

All notable changes to this repository will be documented here.

## 0.1.0

Initial prototype release.

### Added

- FastAPI backend with camera routes, MJPEG streaming, WebSocket updates, and analytics endpoints.
- YOLOv8 OBB parking detector with polygon IoU and temporal smoothing.
- RTSP reader with background frame capture and reconnect behavior.
- React dashboard with live camera stream, slot overlay, stats, slot grid, and calibration wizard.
- PostgreSQL schema and in-memory local fallback.
- Docker Compose files for backend, frontend, Postgres, Redis, and nginx.
- PKLot sample fixture for local testing without a physical camera.
- Proprietary license notice.

## Unreleased
