Sightline uses third-party software and services. Each dependency is governed by its own license and terms.

## Backend

- Python: programming language runtime.
- FastAPI: backend web framework.
- Uvicorn: ASGI server.
- Ultralytics YOLOv8: object detection and oriented bounding box model tooling.
- OpenCV: computer vision and video processing.
- NumPy: numerical computing.
- asyncpg: PostgreSQL client.
- redis-py: Redis client.
- python-multipart: multipart request parsing.
- pytest: Python test runner.

## Frontend

- React: frontend UI library.
- React DOM: browser renderer for React.
- Vite: frontend build tool and dev server.
- @vitejs/plugin-react: Vite React integration.

## Infrastructure

- PostgreSQL: relational database.
- Redis: in-memory data store.
- nginx: reverse proxy and static file server.
- Docker and Docker Compose: local service orchestration.
- FFmpeg: local RTSP sample publishing and media tooling.
- MediaMTX: local RTSP server for demo streams.

## Models And Data

- YOLOv8 OBB model weights are downloaded separately and are not committed to this repository.
- PKLot-derived sample assets are used for local demonstration and review.
